# Start Capgemini

Repositório com exercícios, trilhas e projetos desenvolvidos durante o programa Start Capgemini.

##

## Sobre

Este repositório reúne conteúdos práticos voltados ao desenvolvimento de lógica de programação e construção de aplicações, com evolução progressiva desde conceitos básicos até integração com tecnologias web e Java.

##

## Conteúdos

📌 [Lógica de Programação III – Java](./Lógica%20de%20Programação%20III%20-%20Java)
Exercícios focados em fundamentos da linguagem Java, estruturas de decisão, repetição e manipulação de dados.

📌 [Trilha Java](./TrilhaJava)
Trilha completa integrando lógica de programação, desenvolvimento web e Java, incluindo HTML, CSS, JavaScript, Angular e fundamentos de backend.
