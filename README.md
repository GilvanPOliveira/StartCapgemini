# Start Capgemini

Repositório com exercícios, trilhas e projetos desenvolvidos durante o programa **Start Capgemini**.

##

## Sobre

Este repositório reúne conteúdos práticos voltados ao desenvolvimento de software, com foco na evolução progressiva desde os fundamentos até a construção de aplicações completas.

A estrutura foi organizada para refletir a jornada de aprendizado, passando por lógica de programação, desenvolvimento web e backend com Java.

##

## Conteúdos

[Lógica de Programação III – Java](./Lógica%20de%20Programação%20III%20-%20Java) - Exercícios focados em fundamentos da linguagem Java, incluindo:

* estruturas de decisão
* laços de repetição
* manipulação de dados
* resolução de problemas

##

[Trilha Java](./TrilhaJava) - Trilha completa integrando frontend e backend, com evolução contínua:

* lógica de programação
* HTML, CSS e JavaScript
* Angular
* fundamentos de backend com Java
