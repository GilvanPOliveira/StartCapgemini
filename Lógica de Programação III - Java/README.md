# Lógica de Programação III – Java

Coleção de exercícios desenvolvidos em Java com foco em lógica de programação.

##

## Sobre

Este diretório reúne exercícios voltados ao aprendizado de lógica de programação utilizando Java. Cada pasta representa um exercício independente, com foco em entrada de dados, processamento e saída no console.
