# Lógica de Programação III — Java

Coleção de exercícios desenvolvidos em Java com foco em fundamentos de lógica de programação.

##

## Sobre

Este módulo reúne exercícios práticos voltados ao desenvolvimento do raciocínio lógico utilizando Java.

A estrutura foi organizada em pequenos desafios independentes, abordando desde operações básicas até estruturas de controle, repetição e manipulação de dados.

O objetivo é consolidar os fundamentos essenciais para evolução em desenvolvimento backend.

##

## Objetivo

Desenvolver a base lógica necessária para programação, incluindo:

* operações matemáticas e manipulação de dados
* estruturas condicionais
* laços de repetição
* arrays e estruturas básicas
* resolução de problemas
