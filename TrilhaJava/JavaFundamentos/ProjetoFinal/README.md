# Projeto Final Fundamentos Java

Aplicação de console desenvolvida em Java para integração dos conceitos fundamentais da linguagem.

##

## Sobre

Este projeto reúne os principais conceitos estudados em Fundamentos de Java em uma única aplicação.

A proposta é consolidar o aprendizado por meio de um sistema organizado em módulos, acessado via menu no console.

##

## Objetivo

Integrar conhecimentos fundamentais da linguagem Java, incluindo:

* tipos de dados e operadores
* entrada de dados
* estruturas de controle
* orientação a objetos
* herança e polimorfismo
* organização de código em pacotes

##

## Funcionalidades

### Módulo de Fundamentos

* demonstração de tipos de dados e literais
* cálculo de média e validação de resultados
* leitura de dados via console
* geração de tabuada
* contagem regressiva

##

### Cadastro de Pessoas

* cadastro de pessoas físicas e jurídicas
* uso de herança entre classes
* armazenamento em lista
* listagem de registros cadastrados

##

### Relatório de Mamíferos

* uso de classe abstrata
* implementação de polimorfismo
* cálculo de valores a partir de múltiplas instâncias

##

### Menu Interativo

* navegação entre módulos via console
* controle de fluxo com opções do usuário

##

## Estrutura do Projeto

```text
ProjetoFinal/
├─ app/
├─ fundamentos/
├─ cadastro/
└─ polimorfismo/
```
