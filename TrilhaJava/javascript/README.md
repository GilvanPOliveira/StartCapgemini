# Cadastro de Contatos Avançado

Aplicação web para cadastro de contatos com validação e manipulação dinâmica de dados utilizando JavaScript.

##

## Sobre

Este projeto representa uma evolução de um formulário simples, transformando-o em uma aplicação completa de front-end.

A aplicação permite cadastro, validação, visualização e gerenciamento de contatos de forma dinâmica.

##

## Objetivo

Consolidar conceitos de desenvolvimento frontend, incluindo:

* manipulação de DOM com JavaScript
* validação de formulários
* criação e remoção de elementos dinâmicos
* organização de estrutura HTML, CSS e JavaScript
* construção de interfaces responsivas

##

## Funcionalidades

### Cadastro

* preenchimento de nome, e-mail, cargo e mensagem
* validação de campos obrigatórios
* validação de formato de e-mail

##

### Validação

* exibição de erros em faixa destacada
* mensagens específicas por campo

##

### Pré-visualização

* atualização em tempo real dos dados digitados

##

### Listagem

* inserção de contatos em tabela dinâmica
* visualização dos dados cadastrados

##

### Remoção

* exclusão individual de contatos

##

### Layout

* interface responsiva para diferentes dispositivos
