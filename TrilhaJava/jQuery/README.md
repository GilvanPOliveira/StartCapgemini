# Cadastro de Contatos

Aplicação web simples para cadastro e gerenciamento de contatos utilizando JavaScript e jQuery.

##

## Sobre

Este projeto foi desenvolvido para praticar manipulação de DOM e eventos utilizando jQuery.

A aplicação simula um sistema de cadastro de contatos com formulário, listagem dinâmica e interações no front-end.

##

## Objetivo

Consolidar conceitos básicos de desenvolvimento frontend, incluindo:

* manipulação de DOM
* uso de eventos com jQuery
* validação de formulários
* criação e remoção dinâmica de elementos HTML
* organização de projeto com HTML, CSS e JavaScript

##

## Funcionalidades

### Cadastro de Contatos

* cadastro de nome e e-mail
* validação de campos obrigatórios
* validação de formato de e-mail

##

### Listagem

* exibição dos contatos em tabela
* inserção dinâmica de registros

##

### Filtro

* busca de contatos por nome
* atualização em tempo real

##

### Remoção

* exclusão de contatos da lista
* uso de animações com jQuery
